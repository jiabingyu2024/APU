# Report Preparation Docs

本目录用于汇报阶段，不再记录零散调试过程，而是把 APU + DMA 主线整理成可以答辩、写报告和做 PPT 的材料。

建议阅读顺序：

1. `01_DEFENSE_TECHNICAL_GUIDE.md`：先读这个。它按老师可能追问的方式解释 APU + DMA 主线、六个 final tests、结果字段含义和验收口径。
2. `02_REPORT_APPENDIX_A_TEMPLATE.md`：实验报告长文模板，可直接转成 Word 后继续扩写。
3. `03_PPT_STORYBOARD.md`：PPT 每页内容规划、需要放的图和口头讲法。

关键依据文件：

- final 测试脚本：`dma/final_tests/`
- final 测试结果：`dma/reports/final/`
- DMA RTL：`dma/rtl/`
- 基础 APU RTL：`rtl/`
- Vivado BD Tcl：`dma/vivado/create_apu_dma_project.tcl`
- DMA 设计学习文档：`docs/dma/design/`
